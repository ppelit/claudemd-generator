# 📝 ClaudeMD Generator

ClaudeMD Generator is a lightweight, web-based tool designed to help users create, preview, and format Markdown text effortlessly. Perfect for AI interactions and quick documentation.

## 🚀 Live Demo
[Check out the live app here!](https://ppelit.github.io/claudemd-generator/)

## ✨ Features
* **Real-time Preview:** See your Markdown changes instantly.
* **Clean UI:** Minimalist design for focus.
* **Mobile Friendly:** Works on all devices.
* **No Install Required:** Runs directly in your browser via GitHub Pages.

## 🛠️ Built With
* **HTML5**
* **CSS3**
* **JavaScript**

## 📂 How to Use
1. Open the [live link](https://ppelit.github.io/claudemd-generator/).
2. Type or paste your text into the editor.
3. Copy the generated Markdown for your projects.
